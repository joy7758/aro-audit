# Business Kit

This folder contains practical commercialization assets for `aro-audit`.

## Contents

- `templates/PR_template.md`: integration PR template for partners
- `templates/email_design_partner.txt`: outreach template for design partners
- `templates/release_checklist.md`: repeatable release + evidence checklist

## Positioning

`aro-audit` is positioned as an **Agent Evidence Layer**:
- Open source base: verifiable journal/checkpoint/manifest/verify
- Commercial layer: compliance packs, enterprise key management, proof services

## 12-Month Execution Focus

1. Standardization and packaging (docs consistency, legal readiness, release discipline)
2. Adoption integrations (gateway insertion point, OTEL mapping)
3. Enterprise capabilities (KMS/HSM, key lifecycle, WORM adapters)
4. Proof services (transparency receipt and verification network)
