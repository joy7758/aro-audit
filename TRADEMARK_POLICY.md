# Trademark Policy

## Scope
This policy governs the use of project identifiers, including repository name,
project name, and any future logos or badges distributed by this repository.

## Project Marks
The following identifiers are treated as project marks:
- "aro-audit"
- "AAR/ALC" when used as this project's branded implementation context

## Permitted Use
You may use project marks to:
- Refer to unmodified upstream releases.
- Describe factual compatibility or integration.
- Link to this repository and documentation.

## Restricted Use
You must not use project marks to:
- Imply official endorsement, certification, or partnership.
- Rebrand a modified fork as an official release.
- Use confusingly similar names/logos in commercial offerings.

## Commercial and Press Use
For marketing, press kits, certification claims, or partnership announcements,
contact repository maintainers in advance.

## Third-Party Trademarks
Third-party marks mentioned in this project remain the property of their
respective owners.
